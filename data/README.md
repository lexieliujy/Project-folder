# Data Folder

## Files

- `ACLED Data_MENA_Raw.csv`: exported raw ACLED MENA dataset used as the primary analysis source
- `ACLED Data_MENA_Raw.numbers`: original Numbers-format source file
- `acled_mena_processed.csv`: processed event-level dataset after MENA filtering and NA removal
- `yearly_shift_summary.csv`: yearly comparison of capital-area remote attacks and border-referenced battles
- `spatial_bucket_summary.csv`: event composition across capital areas, border-referenced areas, and other areas
- `country_pattern_summary.csv`: country-level summary statistics

## Processing notes

The processed files are generated by running:

```bash
python3 code/Z_generate_report.py
```

The final analytical sample combines both `Middle East` and `Northern Africa` and removes rows with NA values in the core fields used in the project.
