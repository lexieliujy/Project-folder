## Current policy

As of 2026-04-25, the project keeps only one final report version.

- `docs/index.html` is the only submission webpage.
- Previous v1/v2 and duplicate draft files were removed.
- Future edits should be applied directly to the canonical report baseline.
- Figure 4 in the canonical page uses a yearly map slider, not a cumulative one.

## Update workflow

1. Edit files under `docs/` (and related documentation if needed).
2. Commit and push to `main`.
3. GitHub Pages rebuilds automatically.
